# MyFitnessBuddy

MyFitnessBuddy is a React-based collaborative workout tracker for logging gym sessions, reviewing workout history, tracking lifting progress, comparing stats with a workout partner, and planning shared workouts.

## Team Members

Bruce Le & Dario Bravo

## App Functionalities

- **Dashboard**
  - Shows a quick overview of recent workout activity.
  - Displays weekly stats such as total weight lifted, most frequent exercise, and personal records.
  - Includes shortcuts for logging a workout and viewing progress.

- **Log Workout**
  - Lets users enter a workout title, date, duration, exercises, sets, reps, and weight.
  - Includes example exercise cards for bench press and squat.
  - Shows a workout summary with total sets and estimated volume.

- **Workout History**
  - Displays previous workouts organized by date.
  - Each workout card includes workout type, time, exercises, and a details button.
  - Includes a search field and timeline filter layout.

- **Progress Analytics**
  - Shows strength progress stats such as estimated one-rep max, total volume, and weekly frequency.
  - Includes a visual progress chart, workout frequency heatmap, and muscle focus breakdown.

- **Friends / Compare**
  - Lets users compare workout consistency and strength stats with a friend.
  - Shows side-by-side performance cards and personal record comparisons.
  - Includes activity intensity visualization for social motivation.

- **Shared Workout Plans**
  - Provides a collaborative planning page for shared workouts.
  - Users can view planned exercises, edit entries, and send workout invites.

## How to Run the App

1. Open a terminal and go to the project folder:

```bash
cd "Fitness Tracker/myfitnessbuddy-frontend"
```

2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local URL shown in the terminal. It is usually:

```text
http://127.0.0.1:5173/
```

## Build Command

To create a production build:

```bash
npm run build
```
